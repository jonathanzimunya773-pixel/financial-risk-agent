## 🔧 **TROUBLESHOOTING**

### ❌ ZIP download fails
**Solution:** Use direct link:
```
https://github.com/jonathanzimunya773-pixel/financial-risk-agent/archive/refs/heads/main.zip
```

### ❌ Can't extract ZIP
**Solution:** Try 7-Zip (free):
- Download: https://www.7-zip.org/
- Or use Windows built-in extractor

### ❌ Python not installed
**Solution:**
- Download: https://www.python.org/downloads/
- During install, check "Add to PATH"
- Restart terminal

### ❌ Permission denied (Mac)
**Solution:**
```bash
chmod +x setup.sh
bash setup.sh
```

### ❌ Module import errors
**Solution:**
```bash
pip install -r requirements.txt
```

### ❌ App runs slow
**Solution:** Check your internet (needs data)
